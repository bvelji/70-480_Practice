Content Sizing

Default behavior when box-sizing is not specified or when set to content-box. (Specified in CSS here because Bootstrap overwrites everything with border-box.) Width and Height settings apply to content only. Padding and Border are added to this box.

Left box has content width of 200 px.

Right box has content width of 200 px.

width:200px; height:200px; border: 5px solid black; margin:10px; float:left; border-box:content-box
width:200px; height:200px; border: 5px solid black; margin:10px; float:left; box-sizing:content-box; padding:10px;

Border Box Sizing

Height and width represent content, border, and padding.

Left box has content width of 190px.

Right box has content width of 170 px.

width:200px; height:200px; border: 5px solid black; margin:10px; box-sizing:border-box
width:200px; height:200px; border: 5px solid black; margin:10px; float:left; box-sizing: border-box; padding:10px;

Padding Box Sizing

Only supported in Firefox. Height and width represent content and padding. Border is added to this box.

Left box has content width of 190px.

Right box has content width of 170 px.

width:200px; height:200px; border: 5px solid black; margin:10px; box-sizing:border-box
width:200px; height:200px; border: 5px solid black; margin:10px; float:left; box-sizing: border-box; padding:10px;
